# K-mer Analysis Python Tool

## Desc:
This python script helps analyze DNA squences and outputs the frequency of each k-mer as weel as the nucleotide that follows the k-mers

## Included in this tool
This tool includes a python script cooled script.py (each function of the tool), scripttest.py (to test the tool) and a txt file called input_text.txt. Lastly, this script will create an output.txt file with the validated recorded k-mers.
